Introduction

In this repository, various pieces of source code related to digital historical linguistics will be offered for testing and further development. The programs, most of which are written in JavaScript, mainly serve educational purposes, but some will also be useful for more advanced research in historical linguistics.

Educational Programs

Currently, the following programs are being offered:

  • SoundChanger: A small program with help of which sound change from a source to a target language can be modelled with help of regular expressions. The accurracy of a set of regular expressions is automatically checked and displayed to the user.
  • PhonAlign: A JavaScript implementation of algorithms for sequence alignment and calculation of the edit distance, supporting real-time alignment of two strings using the edit distance, and the alignment of multiple sequence pairs.
  • Diffusion: A very, very simple test of agent-based simulation using Agentscript.
  • Test: Test the current namespace of the DIGHL-library.
  • SAMPA: Test a simple X-SAMPA to IPA converter.
  • MCH: Provides Middle Chinese readings and additional resources for Chinese character input.
